## ⚙️ MCP Configuration — `MCPBasics`

**File:** `MCPBasics` (JSON)

This file configures the **Model Context Protocol (MCP)** server, enabling AI agents to control a real browser via Playwright.

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "@playwright/mcp@latest"
      ]
    }
  }
}
```

### What Each Field Means

| Key | Value | Purpose |
|---|---|---|
| `mcpServers` | Object | Declares MCP server(s) available to the AI host |
| `playwright` | Server alias | Name used to reference this server |
| `command` | `npx` | Launches the server via Node's package runner |
| `args` | `@playwright/mcp@latest` | Always pulls the latest Playwright MCP package |

### How MCP + Playwright Works

```
AI Model (Claude / Gemini / Cursor)
        │
        │  MCP Protocol
        ▼
  @playwright/mcp server
        │
        │  Playwright API
        ▼
  Real Browser (Chromium / Firefox / WebKit)
        │
        ▼
  Target Website
```

The AI model sends high-level instructions (e.g., "click the search box", "navigate to URL") and the MCP server translates them into real Playwright browser automation commands.

### Where to Place This Config

| Tool | Config File Location |
|---|---|
| Claude Desktop | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Cursor | `.cursor/mcp.json` (workspace root) |
| VS Code (MCP ext.) | `.vscode/settings.json` or user settings |

## 🚀 Getting Started with MCP + Playwright

### Prerequisites

- Node.js (v18 or later)
- npm / npx available in terminal

### Step 1 — Install the Playwright MCP Server

```bash
npx @playwright/mcp@latest
```

### Step 2 — Add Config to Your AI Tool

Copy the contents of `MCPBasics` into your AI tool's MCP config file (see table above).

### Step 3 — Restart Your AI Tool

After adding the config, restart Claude Desktop / Cursor / VS Code so it picks up the new MCP server.

### Step 4 — Use It!

You can now instruct your AI to perform browser actions, for example:
- *"Go to google.com and search for Playwright testing"*
- *"Navigate to my app, login with test credentials, and verify the dashboard loads"*
- *"Take a screenshot of the current page"*
